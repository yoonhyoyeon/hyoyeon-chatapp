# hyoyeon-chatapp
> Firebase 기반 실시간 채팅 어플리케이션
## 프로젝트 정보
### Frontend
jQuery
### Backend
Firebase
### Design
Facebook Messenger 디자인을 참고해 개발함.

### 개발 일시
2018.09.20 개발

## 기능
* 모바일 반응형 웹 구현
* 구글, 페이스북 계정 로그인
* 사용자 접속 여부 확인 기능
* 사용자 검색 기능
* 실시간 채팅 기능
